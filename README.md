# Wikipedia-Trending-Topics-ETL

This project collects, processes, and analyzes the most viewed Wikipedia articles daily to identify trending topics over time. It provides insights into which topics gain or lose popularity and can be extended for visualization or further analytics.

#### Features
* Fetches daily top-viewed Wikipedia pages using the Wikipedia Pageviews API
* Stores historical snapshots for analysis
* Aggregates views to identify the most popular articles over time
* Can filter out system pages like "Main Page" and "Special:Search"
* Outputs a clean top articles table ready for visualization or reporting
* 
#### Skills / Technologies
* Python 3.x
* Pandas for data manipulation
* Requests for API calls
* Jupyter Notebook for interactive analysis
